Authentication

Authentication Flow

ขั้นตอนการ login ด้วย LINE account ผ่าน LIFF

1
User เปิด LIFF URL
ใน LINE App ผ่านแชทหรือ Rich Menu
line://app/{liffId}
2
liff.init() รันอัตโนมัติ
LIFF SDK ตรวจสอบ context และเวอร์ชั่น
await liff.init({ liffId: 'xxx' })
3
ได้รับ LINE ID Token
JWT token ที่มีข้อมูลผู้ใช้จาก LINE
liff.getIDToken() // JWT
4
ส่ง ID Token → Backend
ผ่าน endpoint สำหรับตรวจสอบความถูกต้อง
POST /api/auth/line
5
Backend ตรวจสอบ
ตรวจสอบ JWT signature กับ LINE JWK endpoint
GET https://api.line.me/oauth2/v2.1/certs
6
สร้าง/ค้นหา User ใน Planka
เชื่อมโยง LINE user กับ Planka user
UPSERT users (lineUserId, email, name)
7
ออก Planka JWT
สร้าง access token มีอายุ 7 วัน
jwt.sign({ userId, email }, SECRET, { expiresIn: '7d' })
8
เรียก Kanban API
ทุก request ต่อไปส่ง Bearer Token
Authorization: Bearer <planka_jwt>

  • DEL /api/lists/:id

  • GET /api/cards/:id
